Aperçu du code

Ce site est programmé en Javascript Orienté Objet. Il fait appel au framework JQUERY principalement pour la sélection et la manipulation d'objets du DOM. Pour de meilleures performances, les animations de transition sont gérées par la feuille de style CSS3. Voici la liste des classes mises en œuvre.

Objets :

  • SelectionMenu
    Objet instancié : gère un menu qui effectue un filtre. Chaque boutons a donc un état "on" et "off". Dans ce cas-ci, le choix est exclusif. Le menu fonctionne donc de façon analogue à une liste radio: lorsque l'utilisateur opère un nouveau choix, le choix précédent passe en "off".
  • Slideshow_jb
    Objet instancié : une lightbox simple dont le tempo est donné par l'objet timer ci-dessous.
    • Timer_jb
      Objet instancié : un compte à rebours qui peut afficher le temps au format texte (mm:ss:cc) ou sous la forme d'animation contrôlée par javascript.
  • ResponsiveTiles :
    Objet instancié : complète la CSS responsive en basculant le sens du défilement en horizontal lorsque l'écran est en paysage.

Outils:

  • SVGFactory
    Objet statique : crée des graphismes SVG paramétrables.
    • SVGFactory.Params : objet instancié contenant les paramètres d'apparence (taille, graisse, marges) ainsi que des valeurs par défaut.
  • Outils divers :
    Détection, développement et de traitement de tableaux.

Implémentation :

  • Main:
    L'implémentation du code.
  • Navigation:
    Objet statique: gérant la navigation principale.
  • Footer:
    Objet statique: pour le bas de page.
